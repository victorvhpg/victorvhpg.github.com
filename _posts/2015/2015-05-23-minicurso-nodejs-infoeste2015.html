Nos dias 21 e 22 de Maio de 2015, realizei na Infoeste 2015 o Minicurso "Introdução ao Node.js", onde passei um pouco de conhecimento sobre Node.js para a comunidade da FIPP. No curso, passei uma introdução ao Node.js e implementei um chat básico em tempo real com WebSocket.

Abaixo segue os tópicos abordados no curso:

  • Introdução
  • Conhecendo Node.js
  • io.js ?
  • Instalando Node.js
  • Conhecendo Sistema de Módulos
  • Criação de Módulos
  • NPM
  • Arquivo package.json
  • Importância de saber utilizar sistema de controle de versão - GIT
  • Programação Assíncrona (callbacks,Promises)
  • Eventos
  • Streams
  • Módulo http - WEB
  • Express
  • Acesso a Base de Dados
  • Aplicações REST
  • Comunicação em tempo real com WebSockets
  • Chat básico em tempo real
